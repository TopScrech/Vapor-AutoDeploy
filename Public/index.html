This website is running on a Ubuntu VPS. It uses nginx as a http server that reverse proxies to a Swift app using the Vapor framework. nginx is configured to redirect port 80 http requests to port 443 https requests and then to my server side Vapor app on port 8080. nginx handles the SSL certificate.
